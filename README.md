# bSocial
Social Network


## Requirements

- Nodejs
- Running MySQL database

## Tech Stack

**Client:** Vuejs, Vuetify, Bootstrap

**Server:** Node, Express, socket io


## Databse setup
Create database in MySQL with name bsocial
## Server setup
```
cd server 
```
```
npm install
```
```
npm start
```


## Client setup
```
cd client 
```
```
npm install
```
```
npm run serve
```
